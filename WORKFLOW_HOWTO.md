# Workflow How-To Moved

The current end-to-end recipe for AIToolKit workflows driven by AIKit lives in
AIKit [`AGENTS.md`](https://github.com/PangJiaxin0326/AIKit/blob/main/AGENTS.md).
In the standard sibling checkout, use `../AIKit/AGENTS.md`.

This file intentionally no longer repeats the node value algebra, prompt shape,
structured-output advice, auto-bind recipe, caching recipe, or measured numbers.
Those details are maintained in one source of truth so developers do not have to
reconcile stale recommendations across packages.

For API-level details, read the source comments next to the concrete types in
`Sources/AIToolKit/`; for operational guidance, use AIKit `AGENTS.md`.
